# People Counter (IN/OUT)

Projeto simples de contagem de pessoas usando a câmera do celular (web/PWA).
Funciona no GitHub Pages.

## Como publicar
1. Crie um repositório novo no GitHub.
2. Envie **todos os arquivos desta pasta**.
3. Vá em **Settings → Pages**.
4. Source: **Deploy from branch** / Branch: **main** / Folder: **/ (root)**.
5. Aguarde 1–2 minutos e abra o link HTTPS gerado no celular.
6. Permita a câmera e clique em **Iniciar**.

## Dicas
- Use em local fixo, ~2m de altura, levemente inclinado para baixo.
- Ajuste a posição da linha (slider) para a porta/corredor.
- Necessário HTTPS para acesso à câmera.
